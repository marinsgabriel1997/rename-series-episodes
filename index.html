Renomear Episódios

Essa ferramenta automatiza a renomeação dos episódios com base na série, temporada e número inicial informados.
Escolha os arquivos de vídeo que deseja renomear. Cada arquivo corresponderá a um episódio.
Informe manualmente o nome da série ou selecione uma pasta para preencher automaticamente.
Selecione a temporada dos episódios. Isso ajudará a criar um nome com a identificação correta.
Informe o número do episódio a partir do qual a renomeação deve começar. Os arquivos serão renomeados de forma incremental.
A tabela abaixo mostra os nomes originais e os novos nomes gerados. A área de texto exibe os comandos PowerShell que podem ser executados para renomear os arquivos.

Abra o terminal do PowerShell, navegue até a pasta onde os arquivos estão e cole os comandos para renomear os arquivos.
